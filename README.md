# 🌱 GreenScan

An AI-powered sustainability analysis platform that helps users understand the environmental impact of everyday objects.


---

## 🚀 Features

- 🧠 **AI Object Detection (YOLO)**
- 🧊 **3D Visualization (AR-style UI)**
- 🔐 **Blockchain-based Verification (Simulated)**
- 📊 **Sustainability Dashboard**
- 🌍 Carbon Footprint Estimation
- ♻️ Sustainability Scoring
- 🧾 Material Analysis

---

## 🧩 Tech Stack

### Frontend
- React (Vite)
- Three.js / React Three Fiber
- Axios

### Backend
- FastAPI
- YOLOv8 (Ultralytics)
- Python

### Other
- Simulated Blockchain (Hash-based)
- LocalStorage (Dashboard)

---

## 🧠 How It Works

1. User uploads an image
2. YOLO model detects object (chair, bottle, laptop, etc.)
3. System fetches:
   - carbon footprint
   - material info
   - sustainability score
4. Blockchain layer verifies claims (hash-based record)
5. 3D viewer displays object with floating data
6. Dashboard tracks usage and trends

---

## 🔐 Blockchain Implementation

A lightweight blockchain simulation is implemented using:

- Hash generation (SHA-256)
- Timestamped records
- Verification status

This ensures:
- Data integrity
- Tamper resistance
- Transparency
